Video Editing Dataset Sample Naming Specification

1. Naming Convention for Action Categories
    Action categories shall be named using a two-digit numeric code.
      01-Character Replacement
      02-Outfit Change
      03-Background Replacement
      04-Artistic Style Transfer
      05-Big Head Doll
      06-Digital Human
      07-Generation from Frames
      08-Motion Transfer
      09-Image Regeneration
      10-Motion Control
      11-Visual Effects
      12-Stylization
      13-Object Add
      14-Object Remove
      15-Object Replace
      16-Face Swapping
      17-Hairstyle Transform
      18-Body Reshaping
      19-Material Change
      20-Explosion Effect
      21-Fire Effect
      22-Flower Growth Effect

2. Sample Numbering within Categories
    Sample files within each action category directory shall be numbered sequentially starting from 01.
       Example: 0101, 0302

3. File Format Standards
    Video files shall use the .mp4 format.
    Image files shall use the .jpg format.
    Audio files shall use the .wav format.

4. Original Video File Suffix
    Original (source) video files shall use the suffix _raw.mp4.
    Example: 0101_raw.mp4

5. Edited/Generated Video File Suffix
    Edited or AI-generated video files shall use the suffix _edit.mp4.
    Example: 0101_edit.mp4

6. Comparison/Demonstration Video File Suffix
    Videos created for comparison or effect demonstration shall use the suffix _comparison.mp4.
    Example: 0101_comparison.mp4

7. Replacement Image File Suffix
    Source images used for replacement (e.g., in face/character swap) shall use the suffix _target.jpg.
    Example: 0101_target.jpg

8. Start & End Frame Image File Suffixes
    Images representing the first and last frames of a sequence shall use the suffixes _frame_start.jpg and _frame_end.jpg, respectively.
    Example: 0101_frame_start.jpg, 0101_frame_end.jpg
